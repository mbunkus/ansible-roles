Custom facts
=========

This role gathers release information and store them an remote system. 

Requirements
------------

none

Role Variables
--------------

- `custom_facts_templates`(list): filename(s) of templates which should be applied; default: `["deployment.fact.j2", "hotfixes.fact.j2"]`
