Configure Keycloak SAML
=========

This role configures Keycloak as SAML provider.

Requirements
------------

none

Role Variables
--------------

- `configure_keycloak_saml_basedn`(string): The LDAP base dn.
- `configure_keycloak_saml_sp_base_url`(string): The Service Provider base url.
